# Oblivious transfer (OT)

## Definition
A 1-out-of-2 OT Two party protocol between two parties $S$ and $R$, with the following functionality:
```
Shared input: security parameter k
S input: 2 message bits, x[0] and x[1]
R input: 1 bit, b
S output: \bot
R output: x[b]
```
